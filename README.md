## General info
This project is a simple Todo app that can help you to not forget your tasks.

#### Here you can:
* Add todos
* Delete todos
* Update todos
	
## Technologies
Project is created with:
* node version: 14.18.2
* React
* TypeScript
* SCSS
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../todo-app
$ npm install
$ npm start
```
